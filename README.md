# Dadaş Gym - Spor Salonu Yönetim Sistemi

Bu proje, Web Programlama (CPP102) dersi kapsamında geliştirilmiş; frontend tarafında HTML/CSS, backend tarafında Node.js (Express) ve veritabanı olarak MongoDB kullanan Fullstack bir web uygulamasıdır.

## 🚀 Proje Özellikleri
* **Zengin Arayüz:** Özgün CSS ile tasarlanmış responsive arayüz ve süzülen animasyonlu bilgi rozetleri.
* **Dinamik Medya:** Aspect-ratio ile optimize edilmiş YouTube tanıtım videosu ve Google Haritalar entegrasyonu.
* **REST API:** Node.js ile yazılmış API (GET, POST, PUT, DELETE).
* **Oturum Yönetimi:** `express-session` kullanılarak güvenli yönetici girişi.
* **MongoDB Veritabanı:** `populate` (JOIN) ilişkileriyle güvenli veri yönetimi.

## ⚙️ Kurulum ve Çalıştırma
1. Projeyi bilgisayarınıza indirin.
2. Terminali açıp `npm install` komutuyla gerekli modülleri yükleyin.
3. Yerel MongoDB sunucunuzun (localhost:27017) çalıştığından emin olun.
4. `node server.js` komutuyla sunucuyu başlatın.
5. Tarayıcıdan `http://localhost:3000` adresine giderek siteyi görüntüleyin.
